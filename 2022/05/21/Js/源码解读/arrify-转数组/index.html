Array.from arrify

Null 报错：Uncaught TypeError: undefined is not iterable (cannot read property Symbol(Symbol.iterator)) []

作者对于array like处理的回应，翻译一下：故意这么整的，arrify应当只是数组化，而不是对类数组进行转换，对类数组转换可以用Array.from
